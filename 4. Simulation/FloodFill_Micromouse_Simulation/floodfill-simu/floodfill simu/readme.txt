the bit order for floodfill simulator excel files
B00000000
B000VWSEN
V: visted, 1 for visited, 0 for unvisited
N: north, 1 for wall 0 for no wall
E: east, 1 for wall 0 for no wall
S: south, 1 for wall 0 for no wall
W: west, 1 for wall 0 for no wall

You need to store your wall information in such a format otherwise you won't be able to use this simulator
